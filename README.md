## Overview

This project is designed to create attractive and efficient modern NFTs using **React** as the primary framework, combined with **Tailwind CSS** and **Vite**. React offers flexibility in managing and controlling the appearance of NFTs, while Tailwind CSS simplifies styling by eliminating the need for manual CSS. Vite serves as the build tool, optimizing the development process.

## Features

- Responsive and modern design using **Tailwind CSS**
- Fast and efficient development with **Vite**
- Dynamic rendering of NFTs using **React**
- Clean and maintainable code structure

## Tech Stack

- **React 18.0.0**
- **Tailwind CSS** for utility-first styling
- **Vite** for development and build optimization
- **Framer Motion** for animations (if applicable)
- **Radix UI** for accessible component library (if applicable)

## Setup & Installation

---

-   **Clone the repository**

    ```bash
    git clone https://github.com/mochrks/web-nfts-project.git
    cd web-nfts-project
    ```

-   **Install dependencies**

    ```bash
    npm install
    ```

-   **Run the development server**

    ```bash
    npm run dev
    ```

-   The application will be running on `http://localhost:3000`.
